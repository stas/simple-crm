Simple CRM by stas

Simple CRM is a WordPress plugin that lets you define custom fields to extend user profiles and it is also a framework for integration with all kind of CRM API webservices

Inspired from BuddyPress XProfile component. Now it is also a foundation for a framework that adds integration to MailChimp, imports BuddyPress XProfile data, and more... This project was sponsored by Vivanista.com.

Dependencies

WordPress 3.0

Install

Please follow the standard installation procedure for WordPress plugins.
